# AI-Stock 功能头脑风暴（第二轮）

**日期**: 2026-04-01
**方法论**: brainstorm-ideas-new（PM / 设计师 / 工程师 三视角）
**背景**: 基于模块一PRD v2.0 + 用户洞察简报，发散仍未覆盖的增量功能

---

## 产品经理视角

| # | 功能 | 价值点 |
|---|------|--------|
| PM1 | 投研日历视图 | 分析文章+大事提醒+财报日期整合为月历，极强差异化 |
| PM2 | 持仓风险诊断 | 输入持仓，AI诊断行业集中度和风险敞口 |
| PM3 | 同类事件历史回溯 | 分析"美联储加息"时自动引入历史同类事件走势 |
| PM4 | 智能复盘提醒 | 30天后自动推送"你当时分析的准不准" |
| PM5 | 行业分析对比 | 同一行业多次分析后生成横向对比视图 |

## 设计师视角

| # | 功能 | 价值点 |
|---|------|--------|
| D1 | 知识库热力图 | GitHub风格展示每日分析频率，积累感可视化 |
| D2 | 快速笔记模式 | 30秒记录想法，不触发AI，支持后续绑定分析 |
| D3 | 相关文章推荐 | 文章底部展示同行业历史分析，串联知识 |
| D4 | 分析置信度标签 | 保存时标注主观确定性，复盘时看认知偏差 |
| D5 | 进度感成就系统 | 覆盖X行业、积累Y篇、连续Z天，成长可量化 |

## 工程师视角

| # | 功能 | 价值点 |
|---|------|--------|
| E1 | 相似问题检测 | 输入时检索相似历史分析，省API费+驱动复盘 |
| E2 | 结构化数据提取 | 每次分析额外提取股票/行业/影响方向到独立表 |
| E3 | 知识库Markdown导出 | 一键导出，消除锁定顾虑，提升信任 |
| E4 | 每周邮件摘要 | 周一自动发送本周知识库回顾，唤回用户 |
| E5 | 知识库开放API | 供用户接入飞书/企微机器人或自写脚本 |
