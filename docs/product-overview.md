# AI 股票分析平台 — 产品概览

**版本**: 1.1
**日期**: 2026-04-01
**适用对象**: 产品沟通、团队对齐、非技术人员交流
**更新说明**: 新增「产业链分析」第五种事件类型；知识库新增「股票视图」；模块二「相关分析」标签升入MVP

---

## 我们在做一个什么产品？

一句话：**帮A股散户把"看新闻 → 找股票 → 判断买卖"这条链路，在一个工具里跑通。**

现在散户的状态是：
- 看到一条财经新闻，不知道会影响哪些股票
- 分析过的东西，下次就忘了，认知无法积累
- 每天要手动盯着几十只股票，又累又容易漏

我们的产品解决的就是这三个问题。

---

## 产品由三个模块组成

```
┌─────────────────┐    ┌─────────────────┐    ┌─────────────────┐
│   模块一         │    │   模块二         │    │   模块三         │
│  AI事件分析      │───▶│  行情数据展示    │◀───│  策略监控        │
│  & 行业知识库    │    │                 │    │                 │
└─────────────────┘    └─────────────────┘    └─────────────────┘
  看懂新闻，沉淀认知       找到股票，看懂行情       监控自选股，发现信号
```

三个模块互相打通：
- 模块一分析文章里的股票，点击直接跳到模块二看K线
- 模块二的自选股，在模块三自动监控缠论买卖点
- 模块三发现信号时，联动展示模块一的相关历史分析

## 模块一：AI事件分析 & 行业知识库

### 解决什么问题？
> "看到新闻，不知道影响哪只股票"
> "每次分析完，下次又忘了"

### 用户怎么用？

1. **选事件类型**：地缘政治 / 政策法规 / 财报季报 / **产业链分析** / 其他（五选一）
2. **输入事件描述**：比如"中东石油禁运"或"新能源汽车补贴政策"
3. **AI实时生成分析**：文字像打字机一样逐段显示，不用等待
4. **看到结构化结果**（以「产业链分析」为例）：
   - 事件背景
   - 影响逻辑
   - **产业链传导表**（产业链分析专属，最多5层）
   - 受益行业（附推荐股票）
   - 受损行业
   - 风险提示
5. **选择是否保存**：确认行业标签后存入个人知识库（产业链分析会展示所有传导层的行业供勾选）
6. **下次可以搜索**：全文检索历史分析，按行业/时间线/股票三种视图浏览

### 核心亮点

| 功能 | 说明 |
|------|------|
| 流式输出 | 分析结果实时显示，不用盯着转圈等待 |
| 产业链分析 | 新增第五种类型，AI自动追踪事件对上下游产业链的逐层传导（最多5层），含影响程度、时间节奏、推荐股票 |
| 自动生成标题+摘要 | 保存文章时自动生成标题和2句话摘要，知识库一目了然 |
| 行业自动打标签 | AI识别申万行业分类，保存前可手动调整；产业链分析按传导层分组展示所有行业 |
| 三视图知识库 | 行业视图 / 时间线视图 / **股票视图**（按自选股筛选相关文章） |
| 全文搜索 | 按关键词、行业、股票代码搜索历史分析 |
| 相似问题提醒 | 输入问题时提示"3个月前分析过类似问题" |
| 大事提醒 | 提前记录重要会议/政策日期，临近时系统提醒 |
| 新用户引导 | 首次使用有3个示例问题，帮你快速跑通流程 |

## 模块二：行情数据展示

### 解决什么问题？
> "想查一只股票的行情，还要切换到同花顺/东财，太麻烦了"
> "东财广告太多、信息太乱"

### 用户怎么用？

1. **搜索股票**：输入名称或代码（如"宁德时代"或"300750"）
2. **看K线图**：分时 / 日K / 周K / 月K 自由切换
3. **看指标**：均线MA + 成交量（默认显示），MACD / KDJ 可自由开关
4. **看财务数据**：PE、PB、营收、净利润，4个核心指标
5. **看同行对比**：行业内所有股票涨跌/估值/业绩横向排名
6. **管理自选股**：分组管理（重仓股 / 观察股 / 备选股）

### 核心亮点

| 功能 | 说明 |
|------|------|
| 模块一联动 | 分析文章里的股票代码直接点击，侧边栏弹出K线，不跳页 |
| 相关分析标签 | 个股详情页底部展示该股所有历史分析文章（标题+摘要），点击跳回知识库 |
| 行业导航 | 左侧菜单按申万行业分类，一键进入行业股票列表 |
| 行业对比表 | 同行业股票的涨跌幅/PE/PB/营收/利润一览，支持排序 |
| 自选股分组 | 默认3个分组，支持自定义改名和新增 |
| 简洁界面 | 去掉广告和噪音信息，聚焦散户真正需要的数据 |

## 模块三：策略监控

### 解决什么问题？
> "我有十几只自选股，每天要手动一只一只看K线判断买卖时机，太累了"

### 用户怎么用？

1. **把股票加入自选股**（在模块二操作）
2. **系统自动监控**：每天交易时段内，每30分钟自动计算一次
3. **看信号灯**：打开自选股列表，每只股票旁边有颜色标注
   - 🟢 绿灯：出现买点信号，值得重点关注
   - 🔴 红灯：出现卖点信号，注意风险
   - 🟡 黄灯：无明确信号，继续观望
4. **在K线图上看具体位置**：买点是绿色向上箭头▲，卖点是红色向下箭头▼
5. **自己做最终判断**：信号只是参考，用户结合K线形态自行决策

### 核心亮点

| 功能 | 说明 |
|------|------|
| 全自动监控 | 不需要手动操作，交易时段每30分钟自动计算 |
| 信号灯直观呈现 | 一眼扫完所有自选股状态，不用逐一点开 |
| K线箭头标注 | 在K线图上精确标出信号位置，直观可信 |
| 信号历史记录 | 可查看该股过去所有信号，自行评估准确率 |

### 技术说明（给懂技术的人看）
本期实现缠论最基础的两层：**K线合并 + 顶底分型 → 笔识别 → 买卖点**。
后期根据效果逐步扩展到中枢、背驰、一二三类买卖点。

## 三个模块如何协同工作？

### 典型使用场景：早上开盘前

```
1. 打开产品，看到统一首页
   ├─ 左：昨晚保存的AI分析摘要（模块一）
   ├─ 中：自选股列表 + 信号灯状态（模块三）
   └─ 右：今天的大事提醒（模块一）

2. 看到"宁德时代"信号灯变绿
   └─ 点击 → 跳转K线图（模块二）
      └─ K线图底部有绿色▲箭头
         └─ 底部标签"相关分析"有3篇历史文章（模块一）

3. 点开历史文章，看到上周的AI分析
   └─ 分析说"新能源汽车补贴政策直接利好电池行业"
      └─ 结合信号 + 分析，做出今日操作决策
```

### 核心跳转路径

```
模块一（文章中股票代码）
        ↓ 点击
模块二（K线侧边栏弹出）
        ↓ 加入自选股
模块三（自动监控，信号灯显示）
        ↓ 信号出现，点击查看
模块二（K线图箭头 + 相关分析标签）
        ↓
模块一（历史文章，辅助决策）
```

## 产品边界（不做什么）

| 不做的事 | 原因 |
|---------|------|
| 自动交易/下单 | 只是分析工具，最终决策权在用户 |
| 实时行情（Level 2） | 免费数据源15-30分钟延迟，散户够用 |
| AI自动抓取新闻 | 用户主动输入，保证分析质量 |
| 复杂财务分析 | 普通散户用不到杜邦分析等高级指标 |
| 港股/美股 | 先聚焦A股做好 |
| 多人协作 | 个人工具，不做团队功能 |

## 开发节奏规划

### 第一阶段（MVP）— 核心功能跑通

| 模块 | 核心交付 |
|------|---------|
| 模块一 | AI分析输入（5种类型含产业链分析）+ 流式输出 + 保存知识库 + 三视图浏览 + 全文搜索 |
| 模块二 | 个股K线页（日K + 4指标）+ 搜索 + 自选股管理 + **相关分析标签** |
| 模块三 | 顶底分型信号 + 自选股信号灯 + K线箭头标注 |
| 全局 | 模块一→模块二跳转联动 + 模块二→模块一回溯联动 + 统一首页 |

### 第二阶段 — 体验提升

- 模块一：卡片化展示、晚间复盘摘要、持仓风险诊断
- 模块二：行业热力图、详细财务数据
- 模块三：中枢识别、多周期信号、飞书/钉钉推送

## 目标用户画像

| 维度 | 描述 |
|------|------|
| 身份 | A股散户，非专业投资者 |
| 投资风格 | 混合型（波段为主 + 少量长线） |
| 持仓规模 | 同时持有4-10只股票 |
| 时间投入 | 每天1-2小时，早盘前 + 晚上复盘 |
| 使用设备 | 主要PC端使用 |
| 现有工具 | 同花顺、东方财富、雪球等（信息分散） |
| 核心痛点 | 看到新闻不知道影响哪只股票；分析认知无法沉淀 |
