# AI Stock — AI 驱动的 A 股研究平台

> 一款面向个人投资者的 AI 辅助股票研究工具，融合大模型分析、实时行情数据与缠论策略监控三大核心能力。

---

## 项目目标

帮助普通投资者用 AI 的力量做更聪明的投资决策：
- 让 AI 帮你读懂宏观事件对 A 股的影响
- 像专业软件一样查看行情、财务数据
- 用缠论自动提示买卖时机，不再错过机会

---

## 三大核心模块

### 模块一：AI 事件分析 & 行业日志

**功能描述：**
- 输入任意宏观事件（例：美国打伊朗对A股有什么影响？）
- AI 调用大模型（支持 OpenAI / DeepSeek 等，灵活配置）自动生成分析文章
- 文章涵盖：事件背景 → 影响逻辑 → 受益行业 → 值得关注的相关股票
- 用户可选择是否保存该分析
- 保存后自动归类到对应行业日志，形成时间线记录

**行业日志示例：**
```
化工行业
├── 20260331 - 美伊冲突对化工板块影响分析
├── 20260328 - 油价上涨传导到化工成本的逻辑
└── 20260320 - 碳中和政策对精细化工的机遇

军工行业
├── 20260331 - 地缘冲突升温下的军工投资机会
└── 20260325 - 国防预算增加背景下的关键标的
```

**大事提醒功能：**
- 手动添加即将发生的重要事件（如：美联储议息会议、全国两会、财报季）
- 绑定到对应行业，到期前自动提醒
- 提醒时可一键触发 AI 分析

**技术实现要点：**
- 大模型接口：统一抽象层，支持 OpenAI / DeepSeek / 其他兼容接口
- 分析结果结构化存储：行业标签 + 个股标签 + 时间戳
- 数据库：PostgreSQL

---

### 模块二：行情数据展示

**功能描述：**
- 行业列表：展示 A 股所有行业板块，点击进入行业详情
- 行业详情页：该行业所有成分股列表、板块整体涨跌情况
- 个股详情页（类似同花顺/东方财富风格）：
  - 日 K 线图（支持缩放、均线叠加）
  - 分时图（实时价格走势）
  - 基本信息（股票代码、名称、所属行业、市值）
  - 财务数据（营收、净利润、PE、PB、ROE 等关键指标）
  - 关联大事件（来自模块一中标记了该股票的分析文章）

**数据来源：**
- AKShare（免费开源，覆盖全部 A 股数据）
- 数据更新策略：历史数据本地缓存，实时数据按需拉取

**技术实现要点：**
- 图表库：ECharts（K 线图、分时图）
- 财务数据定期同步到本地数据库，减少接口依赖

---

### 模块三：缠论策略监控

**功能描述：**
- 自选股管理：添加/删除关注的股票
- 缠论核心要素自动计算：
  - **笔**：识别价格的顶底结构
  - **线段**：多笔组合形成的趋势段
  - **中枢**：三段重叠区域，判断盘整区间
- 买卖点自动标注：
  - 一买（中枢下方第一个底背驰点）
  - 二买（离开中枢后回调不破前低）
  - 三买（中枢上方突破回踩不入中枢）
  - 对应的一卖、二卖、三卖
- 报警通知：当自选股出现买卖点信号时，系统发出提醒（页面弹窗 + 可选邮件通知）

**技术实现要点：**
- 缠论算法：Python 自实现，基于日线数据
- 后端定时任务：每天收盘后自动计算所有自选股信号
- 前端图表：ECharts K 线图上叠加买卖点标注

## 技术架构

```
前端 (React + Ant Design)
    ↓ HTTP API
后端 (Python FastAPI)
    ├── AI 分析服务 → OpenAI / DeepSeek API
    ├── 行情数据服务 → AKShare
    ├── 缠论计算引擎 → 纯 Python 实现
    └── 定时任务调度 → APScheduler
    ↓
数据库层
    ├── PostgreSQL（结构化数据：分析文章、日志、自选股）
    └── Redis（缓存：行情数据、计算结果）
```

### 技术选型一览

| 层次 | 技术 | 选择理由 |
|------|------|----------|
| 前端框架 | React 18 + TypeScript | 生态成熟，组件丰富 |
| UI 组件库 | Ant Design 5 | 国内常用，金融类界面适配好 |
| 图表库 | ECharts 5 | 支持K线图，文档完善 |
| 后端框架 | Python FastAPI | 轻量快速，AI 生态最佳 |
| 股票数据 | AKShare | 免费开源，A股覆盖全面 |
| 大模型接口 | 统一封装（支持多模型） | 灵活切换 OpenAI/DeepSeek |
| 数据库 | PostgreSQL | 稳定可靠，支持 JSON 存储 |
| 缓存 | Redis | 行情数据高频读取加速 |
| 部署 | Docker Compose | 一键启动，方便迁移 |

## 已知限制与注意事项

1. **AKShare 数据限制**：免费数据有频率限制，实时行情存在几分钟延迟，不适合高频交易
2. **缠论主观性**：缠论本身存在主观判断空间，程序实现为简化版本，仅供参考
3. **投资风险提示**：本工具仅为辅助研究，不构成投资建议，投资有风险
